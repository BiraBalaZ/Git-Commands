Git Commands

Comandos do Git

Aqui iremos abordar os principais comandos do git para iniciar um repositório e commitar os arquivos.

Como Criar um Repositório

1. Abra o terminal na pasta do projeto

Digite o seguinte comando:

git init

Isso irá criar a pasta .git que fica oculta na pasta do projeto.

2. Adicionando os Itens ao Commit

Use o comando git status para verificar as pastas que não foram adicionadas

git status

Isso mostrará os arquivos em vermelho.

git add .

Esse comando com o "." no final adiciona TODOS os arquivos do projeto ao commit. Digitar o comando git status novamente mostrará as pastas, que anteriormente estavam vermelhas, agora na cor verde

3. Colocando a Mensagem do Commit

git commit -m "mensagem do seu commit"

4. Colocando na Branch do Projeto

git branch -M main

5. Estabelecendo Conexão com o Repositório

git remote add origin link_do_repositorio

6. Enviando para o Repositório

git push -u origin main

Como enviar os próximos commits?

Vamos supor que você já tenha um repositório. Esse mesmo, fez o primeiro commit e agora quer fazer alterações e subir elas para o repo. Iremos utilizar quase todos os comandos anteriores:

1. Adicionando os Itens ao Commit

git add .

2. Colocando a Mensagem do Commit

git commit -m "mensagem do seu commit"

3. Enviando para o Repositório

git push
